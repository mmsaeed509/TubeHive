Role Overview

Here you can add your role, such as:

  • • DevOps
  • • Software Engineer
  • • Pentester
  • • etc

Each role comes with its own tools and configurations. When you select a role, you will be able to install the necessary tools, scripts, configuration files, and other resources tailored to that specific role.

For example:


If you select DevOps as your role, it will list all the tools you need, such as Docker, Podman, Kubernetes, Ansible, Terraform, etc.

You will have the option to:

  • • Select all tools or only specific ones.
  • • Choose between a basic installation or a pre-configured setup.
  • • Apply settings tailored to your role.
  • • Roadmap (agenda).
  • • Suggest Docs, articles, courses, etc...
  • • and more.
